## 快速开始

### 启动前端应用

```bash
cd react-app
npm install
npm start
```

应用将在 http://localhost:3000 启动

### 主要功能

- 用户登录和注册
- 病历管理界面
- 实时聊天功能
- 数据可视化
- 跨链操作界面

### 环境配置

确保后端服务运行在 https://localhost:3001

更多详细信息请查看 [完整用户手册](../../backend-app/docs/USER_GUIDE.md)。
